# - 作用：导航到页面顶部 -- 场景：占位符，捕获点击事件
javascript:void(0) - 作用：阻止默认行为，不刷新页面 -- 场景：阻止跳转，配合 JS 使用
"" - 作用：刷新当前页面 -- 场景：需要页面刷新时
about:blank - 作用：打开空白页面 -- 场景：新窗口打开空白页面
rose="button" - 作用：链接表现为按钮，无默认行为 -- 场景：配合 JS 实现按钮功能，无跳转

2、target链接打开方式

_blank 在新窗口或新标签页中打开链接。

新窗口打开

_self 在当前窗口或标签页中打开链接（默认）。

当前窗口(默认)打开

_parent 在父框架中打开链接。

在父框架中打开链接

_top 在整个窗口中打开链接，取消任何框架。

在整个窗口中打开链接，取消任何框架
